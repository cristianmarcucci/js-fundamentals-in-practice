# Task Manager (Vanilla JavaScript)

A simple task manager built with plain JavaScript to practice:

- state management
- DOM manipulation
- localStorage
- accessibility
- clean architecture

## Features

- Add / delete tasks
- Mark tasks as completed
- Filter tasks (all / active / completed)
- Persistent state with localStorage
- Keyboard support (Enter key)

## Concepts practiced

- Data vs UI state
- Single source of truth
- Derived data
- Event handling
- Accessibility basics

## Tech stack

- HTML
- CSS
- JavaScript (ES6+)
